# rlattorra.com

個人ウェブサイト/ポートフォリオサイトのリポジトリです。

## 📋 プロジェクト概要

このプロジェクトは、個人のポートフォリオやプロフィールを紹介するためのウェブサイトです。作品、スキル、経歴などを効果的に発信するためのプラットフォームとして開発しています。

## 🛠️ 技術スタック

- **バックエンド**: PHP
- **その他**: HTML, CSS, JavaScript（予定）

## 📁 プロジェクト構造

```
rlattorra.com/
├── index.php          # メインページ
└── README.md          # プロジェクトドキュメント
```

## 🚀 セットアップ方法

### 必要な環境
- PHP 7.4 以上
- Webサーバー（Apache/Nginx等）

### ローカル環境での実行

1. リポジトリをクローン
```bash
git clone git@github.com:ratorein/rlattorra.com.git
cd rlattorra.com
```

2. PHPビルトインサーバーで起動
```bash
php -S localhost:8000
```

3. ブラウザで `http://localhost:8000` にアクセス
